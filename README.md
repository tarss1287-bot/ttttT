Esta es una página de invitación animada creada con HTML, CSS y GSAP.

## Descripción

El archivo `lolo.html` muestra una tarjeta de invitación con animaciones y botones. Está diseñada para parecer un estilo moderno, similar a las publicaciones de TikTok, con un mini panel de código.

## Cómo usar

1. Abre `lolo.html` en un navegador.
2. Presiona "Sí" o "No" para ver la animación.
